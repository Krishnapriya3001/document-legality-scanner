# Document Legality Scanner

A Python-based tool that analyzes document images using OCR and determines whether a document appears valid or suspicious by detecting legal keywords.

## Features
- Extract text from images using Tesseract OCR
- Detect legal keywords such as agreement, certificate, signature, and date
- Classify documents as VALID or SUSPICIOUS

## Technologies Used
- Python
- OpenCV
- Pytesseract (OCR)
